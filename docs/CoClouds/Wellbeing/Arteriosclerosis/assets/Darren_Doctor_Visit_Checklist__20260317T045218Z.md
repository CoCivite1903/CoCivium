# Darren Family Doctor Visit Checklist
Prepared: 20260317T045218Z

Bring:
- family history summary
- prior cholesterol results
- blood pressure history
- medication list
- smoking status
- diabetes history
- known cardiovascular events in relatives with approximate ages

Ask about:
- lipid panel
- ApoB
- Lp(a)
- inherited lipid disorders
- LDL-C / ApoB targets
- CAC or carotid imaging if useful
- escalation path if targets are not met
- first-degree relative screening
